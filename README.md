# DugonWarden(Nombre provicional)


Sistema de semaforos para mantener tus modulos seguros. Cuando muchos servicios necesitan acceder a un solo modulo puede dañar el hardware o actuar de manera extraña, por eso cree "DugonWarden"
para mantener los modulos seguros, agregando un candado invisible a cada modulo mientras se utiliza.


# Casos de uso

Servicios asincronos que necesitan acceder a un modulo compartido.

# Que resuelve

Resuelve el fastidioso problema cuando se trabaja asincrono o con diferentes servicios con un modulo, haciendolo seguro de acceder sin sobre cargar al modulo por 2 llamadas al mismo tiempo.


# Por que se creo

Principalmente se creo para satisfacer las necesidades de DugonSNAP y sus clientes.
